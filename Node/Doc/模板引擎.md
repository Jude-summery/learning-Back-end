#### 基本概述
模板引擎是一个将页面模板和数据结合起来生成`html`的工具。

#### ejs
一种模板引擎。有3种常用标签：
1. `<% code %>`：运行`js`代码，不输出
2. `<%= code %>`：显示转义后的`HTML`内容
3. `<%- code %>`：显示原始`HTML`内容

> 注意：`<%= code %>` 和 `<%- code %>` 都可以是 `JavaScript` 表达式生成的字符串，当变量 `code` 为普通字符串时，两者没有区别。当 code 比如为 `<h1>hello</h1>` 这种字符串时，`<%= code %>` 会原样输出 `<h1>hello</h1>`，而 `<%- code %>` 则会显示 `H1` 大的 `hello` 字符串。
